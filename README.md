# Guia de referência
> Plano é iniciar com BLAS e LAPACK, posteriormente passar para PLASMA e MAGMA, com exemplos de código para facilitar.

- [BLAS](blas/README.md):

Operações básicas da álgebra linear.

- [LAPACK --Em Construção](lapack/README.md):

Autovalores, autovetores e resolve sistemas lineares.

- PLASMA:

Aprimoramento da LAPACK utilizando GPU's

- MAGMA:

Aprimoramento da LAPACK utilizando GPU's e CPU's
